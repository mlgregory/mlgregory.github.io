HOW TO PLAY WINE GLASSES

This project was a sound and instructional project. First we had to create a instrument, wine glasses, and then create a one minute sound sequence. The next step of this project was to create instructions so any user could replicate the sequence. We made a core case set of instructions as well as an edge case. Both instruction sets include a warm up section, practice section, followed by the sequence instructions. A project for Design Research at MassArt.
